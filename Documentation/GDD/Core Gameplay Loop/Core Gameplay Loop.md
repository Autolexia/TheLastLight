# Core Gameplay Loop
1. **Explore** a region of the corrupted spectrum.
2. **Fight** enemies using melee combat + light effects.
3. **Solve** light-themed traversal/puzzle gates.
4. **Defeat** a region boss (a spectrum-corrupted guardian).
5. **Restore** that band of light → unlock or empower a knight.
6. **Upgrade** gear/skills → venture deeper into darkness.
